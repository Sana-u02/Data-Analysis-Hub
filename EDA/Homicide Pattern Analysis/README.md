# Serial Killer Dataset Analysis

This project examines global serial killer data to uncover trends in victim counts, activity timelines, motives, methods, and geographic patterns. Visualizations and text analysis help reveal behavioral patterns and recurring themes.

## Key Features

* Data cleaning and feature extraction
* Activity duration calculation
* Start and end year trend analysis
* Victim count range comparisons
* Text analysis from case notes
* Frequent word extraction to identify themes

## Main Insights

* Many killers stay active for several years with wide variation in duration.
* Start years cluster in specific historical periods.
* High victim count groups show clear behavioral differences.
* Notes often mention violence, convictions, prison sentences, and women as common victim profiles.
* Punishment related words reflect strong legal actions across many cases.
